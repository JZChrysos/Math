	\section{Historical Overview}
	
	Cantor's Set Theory promised to unify a lot of mathematics and formalize it using the notion of the set. But that foundation was not rigorously defined. For example there was Russell's Paradox, which proposed the set $S$ of all sets not containing themselves, for which $S\in S$ and $S\not\in S$ both lead to contradiction. So formalization was needed.
	
	Multiple formal set theories were proposed, the most prominent of which was initially $\ZF$ (today we use $\ZFC$ most of the time).

	\section{Axioms of Set Theory} 
	
	The language of set theory has only two relations: $\in$ and $=$. $\ZFC$ is a theory in this language generated by the following axioms:
	\begin{itemize}
		\item \textbf{Set Existence}: There exists a set.
		\item \textbf{Extensionality}: $A=B$ iff $x\in A \iff x\in B$.
		\item \textbf{Foundation}: If $A$ has any elements, then it has an element $B$ for which $B\cap A = \emptyset$.
		\begin{itemize}
			\item This implies, for example, that no set has $A=\{A\}$. It is philosophically motivated by the iterative conception of sets, that one can only make $A$ ``after'' one has created all of its elements.
			\item Also note that this does not preclude the existence of infinite descending sequences of sets--it only precludes the existence of a \textit{set} whose elements form an infinite descending sequence of sets.
		\end{itemize}
		\item \textbf{Comprehension}: For every formula $\ph(x)$ in the language of set theory, and given a set $A$, there exists a set consisting exactly of the elements $a\in A$ such that $\ph(B)$ holds.
		\begin{itemize}
			\item Because we begin from a set and filter down, this does not allow the construction of a set of all sets not containing themselves, so Russell's Paradox is avoided.
		\end{itemize}
		\item \textbf{Union}: For all $A$ there is a set consisting of the union of all elements of $A$. 
		\begin{itemize}
			\item So far what we can show is that there is an empty set and there is no set of all sets (easy exercise). But in particular the empty set alone is a model of the axioms thus far.
		\end{itemize}
		\item \textbf{Pairing}: For all $A,B$, there is a set consisting of just $A$ and $B$.
		\begin{itemize}
			\item In particular when $A=B$ this says that $\{A\}$ exists. Now, along with with the fact that the empty set exists, we can create many new finite sets and in particular we can define the successor of any set $A$ to be $S(A):=A\cup \{A\}$, which we now know exists.
		\end{itemize}
		\item \textbf{Powerset}: For any $A$, a set consisting exactly of subsets of $A$ exists.
		\item \textbf{Infinity}: There is a set containing $\emptyset$ which is closed under successor.
		\begin{itemize}
			\item Until this point, none of the preceding axioms guarantee the existence of an infinite set. For example, the Ackermann Model models all axioms of $\ZFC$ except for the axiom of Infinity.
			\item We can now define the set of ``finite ordinals'' where finite means not in bijection with its successor. This thing might not look like $\N$ as we understand it, but it definitely satisfies induction so it is ``good enough.''
		\end{itemize}
		\item \textbf{Replacement}: For any formula $\ph(x,y)$ in the language of set theory that acts like a \textit{function}, one can produce the ``image'' of any set via $\ph$.
		\begin{itemize}
			\item Replacement was not originally in $\ZF$ because there are few situations where it is truly necessary. Whenever you have a set that you know will contain all outputs $y$, you can do the same with comprehension. 
			\item But replacement is needed to show that any well-ordered set is in order-isomorphism with an ordinal, to show the existence of a set that requires infinitely-many invocations of the powerset axiom, and to show Borel determinacy (a result from descriptive set theory).
		\end{itemize}
		\item \textbf{Choice}: For all $A$, there is a set $B$ containing exactly one element of every element in $A$.
		\begin{itemize}
			\item Before Cohen's proof of its independence, it was thought that the axiom of choice might be provable from $\ZF$, but it is not.
			\item Choice is necessary to show that every set can be well-ordered, and even that either $|A|\leq |B|$ or $|B|\leq |A|$ for sets $A,B$.\\
		\end{itemize}
	\end{itemize}

	\underline{Additional proposed axioms independent of $\ZFC$:}
	\begin{itemize}
		\item \textbf{Continuum Hypothesis}: Every uncountable subset of $\R$ has size $|\R|$, i.e. $2^{\aleph_0}=\aleph_1$.
		\item \textbf{Generalized Continuum Hypothesis}: For all $\kappa\geq \aleph_0$, $2^{\kappa}=\kappa^+$.
		\item \textbf{Weakly Inaccessible Cardinal}: There exists $\kappa>\aleph_0$ such that $\kappa=\cf(\kappa)$.
		\item \textbf{Strongly Inaccessible Cardinal}: There exists $\kappa$ such that $2^{\lambda}<\kappa$ for all $\lambda <\kappa$.
		\item \textbf{The Axiom of Constructibility}: Every set is constructible; that is, $V=L$.
		\item \textbf{Martin's Axiom}: 
	\end{itemize}

	\section{Class and Set Models for Set Theory}
	Suppose we want to show that some statement $P$ is consistent with $\ZFC$. The natural way to do this is to give a model. What we'll do is give a ``model'' that is actually a class $\MM$ (i.e. set of propositions) inside of which $\ZFC + P$ holds. \\
%	
%	Then we will have in fact shown $P$ is consistent with $\ZFC$, because if an inconsistency appears, it must appear in $\MM$ as well (exercise).\\

\textit{define absolute, formula relativized to $M$, etc.}\\
	
	\underline{Structures of Set Theory:}
	\begin{itemize}
	\item V, the class of all sets.
	\item WF, the class of \textit{well-founded} ``sets'' considered in $\ZF$ - Foundation.
	\item $R(\kappa)$, the class of \textit{Rank-$\kappa$} sets, is defined inductively as follows: $R(0)$ is just the empty set, and in general $x\in R(\kappa)$ iff all elements of $x$ have rank less than $\kappa$.
	\begin{itemize}
		\item This is a set, and it can be constructed fairly simply by recursion.
		\item If $\kappa$ is strongly inaccessible, $|R(\a)|<\kappa$ for all $\a<\kappa$.
	\end{itemize}
	\item $H(\kappa)$, the class of \textit{Hereditarily Cardinality-$\kappa$} sets, is the class of sets $x$ such that the ``transitive closure'' of $x$ has cardinality less than $\kappa$.
	\begin{itemize}
		\item Whereas the class of sets with cardinality $<\kappa$ is not a set, $H(\kappa)$ \textit{is} actually a set; to show this, we can show that $H(\kappa)$ is contained in $R(\kappa)$, a set, thus by Comprehension it must be a set.
		\item The proof: if $t$ is the transitive closure of $x$, let $\a$ be the least ordinal that is not the rank of anything in $t$. Then for $\b>\a$, $\b$ cannot be the rank of anything in $t$ either, since $t$ is transitively closed and thus $\a$ would have to be the rank of some of its elements. So the ranks of elements in $t$ are all less than $\a$, so $x$ itself has rank at most $\a$. 
	\end{itemize}
	\item OD, the class of \textit{ordinal-definable} sets, i.e. sets $A$ such that $A$ is \textit{definable} in $\ZFC$ using finitely many ordinal parameters within some $R(\b)$ (the notion of definability means a specific thing). 
	\begin{itemize}
		\item This includes every set $A$ which is ``definable'' in the usual sense, i.e. there is some formula $\ph(x)$ (possibly with parameters) which is true iff $x=A$.
		\item The class OD itself can also be described by some formula $\Phi(\a,x)$ in the sense that every $A\in OD$ has some ordinal $\a$ such that $\Phi(\a,x)$ iff $x=A$.
		\item Also $V=OD$ is consistent, though $OD$ is not transitive.
	\end{itemize}
	\item HOD, the class of \textit{hereditarily} ordinal-definable sets, i.e. sets $A$ such that the transitive closure of $A$ is in OD.
	\item $L$, the class of \textit{Constructible Sets}.
	\begin{itemize}
		\item Let $\D(A)$ be the ``definable powerset'' of $A$. Properties:
		\begin{itemize}
			\item $|\D(A)|=|A|$ for all infinite $A$.
			\item Every finite $x\subseteq A$ is in $\D(A)$.
			\item If $A$ is transitive then $A\subseteq \D(A)$.
		\end{itemize}
		\item $L$ can be defined as the smallest class of sets containing $\emptyset$ and closed under (any ordinal number of applications of) $\D$.
		\begin{itemize}
			\item $L$ has the amazing property that for all transitive proper class models $M\models \ZF - Pow$, $L = L^M$; that is, what $M$ calls $L$ is in fact $L$.
			\item For transitive set models $S\models \ZF- Pow$, $L(o(M))=L^M$ (where $o(M)$ is the least ordinal not in $M$).
			\item Note that $L\subseteq HOD$, so $V=L$ implies $V=HOD$.
		\end{itemize}
		\item $L(\a)$ is the class of sets which can be reached within $\a$-many applications of $\D$.
		\begin{itemize}
			\item (AC) $|L(\a)|=|\a|$ in general.
			\item If $\kappa$ is weakly inaccessible, $L(\kappa)\models \ZFC+ (V=L)$. For regular cardinals $\kappa$, $L(\kappa)$ doesn't necessarily model powerset.
		\end{itemize}
	\end{itemize}
	\item $M$, the ``ground model,'' a countable transitive model of $\ZFC$.
	\begin{itemize}
		\item Obtained by repeatedly adding one set for every existential in $\ZFC$ and then taking Mostowski collapse.
	\end{itemize} 
	\end{itemize}

	\section{Consistency Results}
	G\"odel showed that $\Gamma\nvdash \Con(\Gamma)$ for any fragment of set theory $\Gamma$. We can therefore define the partial order $\lhd$ on fragments, where 
	$$\Gamma \lhd \Lambda \; \iff \; \Lambda \vdash \Con(\Gamma).$$ 
	With this notation, G\"odel's second incompleteness theorem says that $\neg(\Gamma \lhd \Gamma)$. Showing $\Gamma \lhd \Lambda$ will typically involve working within $\Lambda$ to produce a model of $\Gamma$.
	
	We also define a finer partial order $\leq$ by 
	$$\Gamma\leq \Lambda \; \iff \; \Con(\Lambda)\implies \Con(\Gamma)$$
	and say $\Gamma \sim \Lambda$ if $\Gamma \leq \Lambda$ and $\Lambda \leq \Gamma$.\\
	
	\textit{list some properties}\\
	
	\textit{set vs class}\\
	
	\underline{Consistency Results:}
	\begin{itemize}
		\item $\ZF$ - Inf $\leq \ZF$ - Fou:
		\begin{itemize}
			\item $\ZF$ - Fou proves that $R(\omega)$ is a model of $\ZF - Inf + \neg Inf$.
			\item $\ZF$ - Fou also proves that $WF$ is a model of $\ZF$.
		\end{itemize}
		\item $\ZFC$ - Pow $\lhd$ $\ZFC$: 
		\begin{itemize}
			\item For a regular cardinal $\kappa>\omega$, $H(\kappa)$ is a model for $\ZFC$ - Pow. 
			\item $H(\kappa)$ can only model Powerset if $\kappa$ is strongly inaccessible, equivalently $H(\kappa)=R(\kappa)$. This shows that $\ZFC$ cannot prove the existence of such a $\kappa$, since it cannot prove its own consistency.
		\end{itemize}
		\item $\ZFC$ - Pow + $\neg$Pow $\lhd$ $\ZFC$:
		\begin{itemize}
			\item This is modeled by $H(\omega_1)$, for example. This also shows that $\ZFC$ - Pow cannot prove the existence of an uncountable set.
		\end{itemize}
		\item $\ZFC$ - Rep $\lhd$ $\ZFC$:
		\begin{itemize}
			\item $R(\omega+\omega)$ is model for $\ZFC$ - Rep.
		\end{itemize}
		\item $\ZFC$ $\lhd$ $\ZFC$ + SIC:
		\begin{itemize}
			\item If a strong inaccessible $\kappa$ exists, then $R(\kappa)=H(\kappa)$ is a set model for $\ZFC$.
		\end{itemize}
		\item $\ZFC\sim \ZF$:
		\begin{itemize}
			\item We use the class HOD. It suffices to prove $HOD\models \ZFC$ in $\ZF$. The key axioms to be proven are comprehension and choice. 
			\item For Comprehension, let $\ph(x)$, be a formula, $y\in HOD$. Let $A := \{x\in y:\ph(x)\}$, and we seek to show $A\in HOD$. $\ZF$ already has comprehension, so we get that $A$ is a set. We can also immediately see that it is definable, so because $A\subset y$, $y\in HOD$, $A\in OD$, we see that $A\in HOD$ as well, by definition of HOD.
			\item For Choice, it is equivalent to show the well-ordering principle.
		\end{itemize}
		\item $\ZF+ (V=L) \sim \ZF$:
		\begin{itemize}
			\item $L$ itself is a model of $\ZF+ (V=L)$, though this is not trivial.
		\end{itemize}
		\item $\ZFC + GCH \sim \ZF$:
		\begin{itemize}
			\item $\ZF$ proves that $L$ is a class model of $\ZFC + GCH$. Choice is fairly easy; just note that $V=L\implies V= HOD$ and HOD has Choice.
			\item For GCH, one can show in $\ZF$ that $\PP(L(\kappa))\subseteq L(\kappa^+)$, which implies GCH.\\
		\end{itemize}
	\end{itemize}
	
%	$\ZFC$ can't prove that any given set models all axioms of $\ZFC$, as this would be a proof of $\Con(\ZFC)$. However, if $\ZFC$ is consistent, $\ZFC$ can produce a set in which any given finite subset of the axioms of $\ZFC$ hold. And in fact, for any finite set of $\ZF$ sentences $\ph_1,\dots,\ph_n$, for sufficiently large ordinals $\a$, $\ZF$ proves that $R(\a)$ is a model for $\ph_1,\dots,\ph_n$, and similarly for $H(\a)$, or any stratification of $V$ by ordinals.\footnote{This implies that $\ZFC$ is not finitely axiomatizable.} With choice, we can extend any $X$ to model $\ZF,\ph_1,\dots,\ph_n$ with the same size as $X$ (or of countable size, if $X$ is finite). So in particular, $\ZFC$ can produce a countable model $M$ of any finite subset of itself. Then one can take the Mostowski collapse of $M$ to get a transitive countable model within any transitive class.

\subsection{Cohen's Proof of Consistency of $\ZFC + \neg CH$} Given any $M$-generic $G$, we can construct a transitive model $M[G]$, called a ``generic extension of $M$,'' which is the ``$\ZFC$ closure'' of $M\cup \{G\}$. We can define a forcing language and relation $\Vdash$ and so on to characterize $M[G]$ by the forcing theorem: $M[G]\models \sigma \iff \exists p\in G: \; p\Vdash \sigma$.

	\section{Infinitary Combinatorics}
	
	Fixing some $\kappa \geq \omega$, say that two subsets $x,y\subseteq \kappa$ are ``almost disjoint'' if $|x\cap y|<\kappa$. An \textit{almost disjoint family} is a set of subsets of $\kappa$ which are pairwise a.d. and all have size $\kappa$. It is maximal if there is no way to add another subset of $\kappa$ while maintaining this property. The immediate question is: how large can these families get?\\
	
	If we were working with ``disjoint'' rather than ``almost disjoint,'' the largest d.f. has size $\kappa$ (just map $\kappa$ to $\kappa\times \kappa$ and take the rows). But for a.d.f., the answer is more complicated.\\
	
	\begin{itemize}
		\item If $\kappa$ is regular, the largest a.d.f. is size at least $\kappa^+$.
		\item Is there a maximal family of size exactly $\kappa^+$?
		\begin{itemize}
			\item This would follow from GCH but is independent of $\ZFC$.
		\end{itemize} 
		\item Is there a maximal family of size exactly $2^{\kappa}$?
		\begin{itemize}
			\item For $\kappa=\omega$, yes. (exercise).
			\item For larger $\kappa$, it is true if $2^{<\kappa}=\kappa$.\\
		\end{itemize}
	\end{itemize}
	
	A family of sets is a $\Delta$-\textit{system} if every pairwise intersection is equal to the same ``root'' $r$.\\
	
	\begin{itemize}
		\item If $\kappa <\theta$ are cardinals such that $\theta$ is regular, then for every $S$ of cardinality $\theta$ containing elements of size less than $\kappa$, there is some $F\subseteq S$ with $|F|=|S|=\theta$ which is a $\Delta$-system.\\
	\end{itemize}
	
	A topological space $X$ has the \textit{countable chain condition} if there is no uncountable family of pairwise disjoint \textit{open} subsets of $X$.
	
	\begin{itemize}
		\item Equivalent to say that there is no countable descending chain of open subsets whose closures are also strictly descending.
		\item If $X$ and $Y$ have c.c.c., does $X\times Y$?
		\begin{itemize}
			\item Independent!
			\item It is equivalent to show that arbitrary products preserve c.c.c. (proof using delta system lemma to reduce to finitely many coordinates).\\
		\end{itemize}
	\end{itemize}

	\section{Forcing and Martin's Axiom}
	
	In the absence of CH, we have to wonder: how do cardinals between $\omega$ and $2^{\omega}$ behave? Are they like $\omega$ or like $2^{\omega}$? (for example, is $2^{\kappa}=2^{\omega}$, or is it greater?) Martin's axiom comes in to say ``they are like $\omega$.''\\
	
	To define Martin's Axiom precisely requires a definition of forcing. A notion of forcing $P$ is a partial order of ``conditions'' on sets such that a descending chain of conditions always has a set witnessing the whole chain. A set of conditions $D$ is \textit{dense} if every condition can be extended to a condition in $D$. 
	
	For every countable family $\D$ of dense conditions there is an object obeying all of $\D$ (meet each $D\in \D$ at a finite stage, get a countable descending chain, and you have a witness). But what about an uncountable $\D$? If $|\D|=2^{\omega}$ then the answer must be no, since $2^{\omega}$ is also the size of the space of sets we are constructing; otherwise we could give conditions saying ``avoid this set'' for every set, creating a set that doesn't exist. But for $|\D|=\kappa \in [\omega,2^{\omega}]$ we have a genuine question.\\
	
	\textit{explain importance of c.c.c.}\\
	
	\textbf{Martin's Axiom on $\kappa$}: If $P$ is a partial order with the c.c.c. and $\D$ is a dense family of size $\leq \kappa$, then there is a $\D$-generic filter.\\
	
	\underline{Applications of Martin's Axiom on $\kappa$:}
	 \begin{itemize}
	 	\item There is a subset $A\subset \R^2$ meeting every line in exactly 2 points.
	 	\item There is no maximal a.d.f. $A\subset 2^{\omega}$ of size $\kappa$.
	 	\begin{itemize}
	 		\item Equivalent to show that given an a.d.f. $A$ of size $\kappa$, we can add another set to $A$. To show this we develop a notion of forcing where conditions are $(S,F)$ where $S\subseteq \omega$ and $F\subseteq A$. $(S,F)$ is interpreted on $\tt{G}$ as saying $S\subseteq \tt{G}$ and $\mathtt{G}\cap X\subseteq S$ for all $X\in F$.
	 	\end{itemize}
	 	\item The union of $\kappa$-many measure-0 sets in $\R$ is measure-0.
	 	\begin{itemize}
	 		\item Let $\{M_{\a}\}_{\a<\kappa}$ be a collection of null subsets, and let their union be $M$. We construct a measurable cover of $M$ with measure $<\ep$ by forcing. We take $P$ to be the open subsets of $\R$ with measure at most $\ep$, with extension defined by $p\leq q \iff q\subseteq p$, with $p$ interpreted as $\mathtt{G}\subseteq p$. 
	 	\end{itemize}
	 	\item $2^{\kappa}=2^{\omega}$.
	 \end{itemize}
